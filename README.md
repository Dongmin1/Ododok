# 오도독 홈페이지 V2.4

V2.3.2를 기준으로 첫 화면의 균형과 미세 인터랙션을 다듬은 마감 버전입니다.

## 변경 사항
- 노트북 높이에서 다음 콘텐츠가 20~30px 정도만 보이도록 첫 화면 비율 조정
- Hero 배경에 매우 은은한 크림 그라데이션 적용
- Hero 사진 위치를 소폭 보정하고 움직임을 완화
- 스크롤 시 사진에 1.2% 이내의 미세 확대 적용
- 헤더 로고 소폭 확대
- 메뉴와 햄버거 크기 미세 조정
- Philosophy 구분선과 아이콘 비율 정리
